# Snow Depth Explorer

Static site for interactive and animated snow-depth comparison charts.

## Pages
- `index.html` (home)
- `snow_depth_explorer.html` (Mt. Shasta / MSSKI)
- `tahoe_donner_explorer.html` (Tahoe/Donner / CSSL)
- `chart_animation_player.html` (auto animator)
- `chart_date_scrubber.html` (manual date scrubber)

Configured for GitHub Pages custom domain: `calweathergeek.com`.
